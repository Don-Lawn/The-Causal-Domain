## Z

### **Zero‑Phase Reference (Gauge Zero)**

**Definition**

 The arbitrary choice of where phase angle \phi  is set to zero.
 It is a gauge choice with no physical significance.

**Conceptual Role**

 Zero‑phase reference expresses the gauge freedom of the phase plane.
 Changing the zero point does not alter the geometry, the Q‑axis, or any physical observable.
 Only phase *differences* matter.

**Geometric Expression**
$$
\phi \rightarrow \phi +\phi _0
$$


with no change to \(Q_c\) or the phase plane.

**Cross‑links**
 Gauge Freedom, Phase Rotation, Phase Plane, \(Q_c\).

**Interpretive Summary**
 Zero‑phase reference is a bookkeeping choice.
 It sets the origin of \(\phi\)  but carries no geometric or physical meaning.
 The geometry depends only on the orientation of the phase plane, not on where \phi  happens to be zero.
