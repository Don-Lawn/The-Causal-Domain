There are several ways of approaching understanding of it.

Firstly, there is a Glossary attempting to explain each concept in isolation, for use as a reference guide.

[Glossary](../glossary/index.md)

Secondly, there is a sequence showing how we discovered different parts of it, in chronological sequence.

[The Discovery Sequence](TheDiscoverySequence.md)

Thirdly, there is a sequence that presents problem, solution, and consequences in logical sequence, but it may be too abrupt for many readers. It does follow the more rigorous scientific approach.
